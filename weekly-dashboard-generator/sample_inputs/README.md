# Sample Inputs

This folder is for sanitized, generic sample files only.

Do not store real client exports, project plans, transcripts, logos, generated
dashboards, names, email addresses, or other confidential information here.
Project managers should upload working files through the Streamlit app.

`dashboard_sample.json` contains fictional data used for local demonstrations
and automated tests. Raw CSV, Excel, transcript, and image files in this folder
are ignored by Git as an additional safeguard.
